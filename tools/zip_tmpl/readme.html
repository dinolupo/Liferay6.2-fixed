Remember Liferay Portal is built by and for a very large open community. Be sure to participate by contributing back to the Liferay Community.

I. New Features

Liferay Portal 6.2 includes many new features targeting end users (power users), systems administrators and developers for enterprise deployments. Each area includes multiple features designed to support the most complex deployments. Liferay includes a web content management and document repository with multiple collaboration services all wrapped up in a portal framework. These features combine to allow Liferay to support an extremely broad range of use cases. See the full list of new features here on the Liferay 6.2 New Features Wiki Page.

II. Community Resources

Find information, participate in discussions and contribute back to the community at: Liferay.org. Be sure to "Like" Liferay on Facebook to collaborate with others and to find people in your field or specialty who are also using Liferay. Also be sure to "Follow" us and the community on Twitter which uses TweetRay to repost all tweets from the community who use the tag #Liferay.

  1. Documentation and Installation
    1. A quick installation guide is available on the Liferay Wiki.
    2. Official Documentation for Liferay is available on liferay.com as well as from online publishers.
    3. Liferay in Action is available from Manning Publications and includes a definitive review of many Liferay features and capabilities.
  2. Liferay's Open Source Community
    1. Go to liferay.org to search for answers within the forums, to post answers for others, and to find online reference materials in the Wiki section.
  3. Bug Reporting
    1. While the Liferay Community cannot ensure all reported bugs will get fixed or receive a response right away, contributing bug information to Liferay supports everyone and is one of the most valuable ways you can contribute back to the Liferay Community. If you find a bug, please log as much information as possible into a new ticket at: http://issues.liferay.com.
  4. Suggesting New Features
    1. The Liferay open source project relies on its community for many of the evolutionary and revolutionary ideas that eventually find their way into a release. Liferay strongly believes in empowering the community not only to be able to create change through feature requests but also to feel responsible for socializing, recommending, and implementing their own and others' ideas. If you have a great idea for Liferay, check out the Feature Request Dashboard, which provides an easy way to create, view, and recommend feature requests, and help steer the direction of Liferay!
  5. Liferay Marketplace
    1. The Liferay Marketplace is an exciting new hub for sharing, browsing and downloading Liferay-compatible applications. As enterprises look for ways to build and enhance their existing platforms, developers and software vendors are searching for new avenues to reach this market. Marketplace leverages the entire Liferay ecosystem to release and share apps in a user-friendly, one-stop site. Browse the available apps, or visit the developer resources pages to learn how to build your own apps!
